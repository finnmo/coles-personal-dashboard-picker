# Household Dashboard

A wall-mounted iPad grocery dashboard. Shows your household grocery items sorted by how overdue they are for restocking. Tap a product to mark it as bought and add it to a shared shopping list.

Runs in Docker on a home server. Password-protected. Light and dark mode.

---

## How it works

**Dashboard** — Products are sorted by a priority score (`days since last purchase ÷ repurchase interval`). Overdue items bubble to the top with a red badge. Items you've never bought sit at the bottom. Tap any tile to record a purchase and send it to your shopping list.

**Shopping list** — A floating button opens a bottom sheet showing everything you've added this session. Swipe left on any item to remove it, or clear all at once.

**Add products** — Tap "Add" in the header to search the Open Food Facts catalogue and add products to your dashboard. Set a repurchase interval (e.g. 7 days for milk, 30 for olive oil) and the priority score handles the rest.

**Idle dim** — After 5 minutes without interaction the screen dims to protect the display. Tap anywhere to wake.

## Setup

### Prerequisites

- Node 20+, Docker + Docker Compose
- For Apple Reminders: an iOS Shortcut set up per [docs/SHORTCUTS_SETUP.md](docs/SHORTCUTS_SETUP.md)
- For Google Tasks: OAuth credentials per `npx tsx scripts/google-oauth-setup.ts`

### Running locally

```bash
git clone <repo> && cd coles-personal-dashboard-picker
npm install

cp .env.example .env
# Edit .env — see required values below

mkdir -p data
npx prisma migrate deploy
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and log in with your password.

### Running in Docker

```bash
cp .env.example .env
# Edit .env

docker compose up -d
```

The container runs migrations automatically on start. Data is persisted in a named Docker volume.

## Environment variables

| Variable                | Required  | Description                                                              |
| ----------------------- | --------- | ------------------------------------------------------------------------ |
| `DATABASE_URL`          | Yes       | SQLite path — use `file:/app/data/dashboard.db` in Docker                |
| `APP_PASSWORD_HASH`     | Yes       | bcrypt hash — run `npx tsx scripts/generate-password-hash.ts <password>` |
| `SESSION_SECRET`        | Yes       | 32+ random characters                                                    |
| `SESSION_EXPIRY_HOURS`  | No        | Session lifetime (default 24)                                            |
| `LIST_PROVIDER`         | Yes       | `apple_reminders` or `google_tasks`                                      |
| `APPLE_SHORTCUTS_NAME`  | If Apple  | Name of your iOS Shortcut                                                |
| `GOOGLE_CLIENT_ID`      | If Google | OAuth client ID                                                          |
| `GOOGLE_CLIENT_SECRET`  | If Google | OAuth client secret                                                      |
| `GOOGLE_REFRESH_TOKEN`  | If Google | Generated by `google-oauth-setup.ts`                                     |
| `GOOGLE_TASK_LIST_ID`   | If Google | Target task list ID                                                      |
| `OPEN_FOOD_FACTS_EMAIL` | No        | Your email, included in the User-Agent sent to Open Food Facts           |
